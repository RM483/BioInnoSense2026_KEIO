# ⑧ UIデザイン仕様

## 1. デザイン原則
Apple水準の静的な美しさ: 大きな余白、カードUI、少ない色数、写真を主役に。
高齢者にも読める: 本文16sp以上、主要数値は40sp+、タップ領域48dp以上、WCAG AAコントラスト。

## 2. カラーパレット (白/青/グレー + アクセント)

| トークン | 値 | 用途 |
|---|---|---|
| `surface` | #FFFFFF | 背景 |
| `surfaceContainer` | #F5F7FA | カード背景/セクション |
| `primary` | #2563EB (青) | CTA、アクティブ、グラフ線 |
| `onSurface` | #1A1D21 | 本文 |
| `onSurfaceVariant` | #6B7280 (グレー) | 補助テキスト |
| `outline` | #E5E7EB | 罫線 |
| `accent` | #F59E0B (琥珀) | 警告・H2高値のみ |
| `error` | #DC2626 | エラー |

ダークモードはMaterial3 `ColorScheme.fromSeed(seedColor: #2563EB, brightness: dark)` で自動導出。

## 3. タイポグラフィ
| スタイル | サイズ/ウェイト | 用途 |
|---|---|---|
| displayLarge | 56 / w700 | 測定中の現在値 (ppm) |
| headlineMedium | 28 / w700 | 画面タイトル |
| titleMedium | 18 / w600 | カード見出し |
| bodyLarge | 16 / w400 | 本文 (最小サイズ) |
| labelLarge | 16 / w600 | ボタン |

## 4. コンポーネント
- **カード**: 角丸20、影なし(枠線 `outline` 1px)、内側padding 20。
- **主ボタン**: 高さ56、角丸28、`primary`塗り、白文字。
- **数値表示**: 等幅タブラー数字 (`FontFeature.tabularFigures`)。
- **グラフ**: 線 `primary` 2.5px、面グラデーション(primary 20%→0%)、閾値ラインは`accent`破線。

## 5. 画面ワイヤーフレーム(要点)

```
ホーム                          測定
┌──────────────────┐   ┌──────────────────┐
│ こんにちは、茂木さん      │   │ ← ポチ  ● 接続中     │
│ ┌──────────────┐ │   │                      │
│ │ 🐶写真  ポチ       │ │   │      12.4 ppm        │ ← displayLarge
│ │ 柴犬・4歳・8.2kg   │ │   │   基準内 ✓ (青)      │
│ └──────────────┘ │   │ ┌──────────────┐ │
│ 最新測定  8.1ppm 平常  │   │ │ リアルタイムグラフ │ │
│ ┌──────────────┐ │   │ │   (fl_chart)      │ │
│ │  ▶ 測定をはじめる  │ │   │ └──────────────┘ │
│ └──────────────┘ │   │ 経過 02:31  平均 10.2 │
│ [履歴] [食事] [症状]   │   │ ┌───■ 停止 ───┐   │
└──────────────────┘   └──────────────────┘
```
- ホーム: 犬写真カードが最上部で主役。CTAは1つだけ大きく。
- 測定: 現在値を最大要素に。異常フラグ付きサンプルはグレー点で描画。
- 履歴: 日付グルーピングのリスト+各行にスパークライン。タップで詳細(全系列+食事/症状の重畳)。
- BLE接続: スキャン中はパルスアニメ、電波強度をバー表示、接続済みデバイスを上部固定。
- エラー: イラスト+一文+具体的な次の行動ボタン(「再接続する」等)。技術用語を出さない。
